## Description

HooBank is a modern business card website designed for a bank. The website serves as an online presence for the bank, providing information about its services, products, and contact details. It features a contemporary design and is implemented using React, Vite, and Tailwind CSS.

## Features

- **Contemporary Design**: A modern and sleek design to represent the bank's brand.
- **Informative Content**: Provides information about the bank's services, products, and contact details.
- **Responsive Layout**: Ensures optimal viewing experience across all devices.
- **Built with React**: Utilizes React for building dynamic user interfaces.
- **Fast Development**: Developed with Vite for quick and efficient development.
- **Styling**: Styled with Tailwind CSS for rapid UI development.

## Tech Stack

- **React**: Frontend library for building user interfaces.
- **Vite**: Build tool for modern web development.
- **Tailwind CSS**: Utility-first CSS framework for styling.
